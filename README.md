One Good Deed | عمل خير واحد 🌙

One Good Deed is a minimalist Android application designed to encourage daily acts of kindness. In a world that moves fast, this app serves as a silent companion, suggesting one simple, meaningful good deed every day to help build a habit of compassion.

✨ [Visit the Live Website](https://3elwa00.github.io/one-good-deed/)

🚀 [Quick Download](https://github.com/3elwa00/one-good-deed/releases/download/apk/1GoodDeed.apk)

You can always find the latest version on our [Releases Page](https://github.com/3elwa00/one-good-deed/releases/).

✨ Features

📅 Daily Inspiration: Receive a hand-picked random good deed every day.

🌍 Bilingual Support: Full support for both English and Arabic interfaces.

🔒 Privacy First: Zero tracking, no ads, and no data collection.

⚡ Lightweight: Fast, simple, and works offline.

❤️ Open Source: Built with love for the community.

🛠 Tech Stack

App: Android (Kotlin / Java)

Web: Tailwind CSS, Glassmorphism UI, Responsive Design.

🤝 Contributing

This project is open-source! If you have suggestions for new "Good Deeds" or want to improve the code, feel free to fork the repo and submit a pull request.
